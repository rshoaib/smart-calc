# Model Selection Quick Reference

Before starting a conversation, select the correct model from the dropdown under the chat box.

## Task → Model Mapping

| Task Type | Model | Why |
|-----------|-------|-----|
| UI/UX, layouts, components | Gemini 3 Pro High | Best at visual design |
| CSS / Tailwind styling | Gemini 3 Pro High | Strong at styling |
| SEO, meta tags, sitemap | Gemini 3 Pro High | Web-aware |
| AdSense compliance | Gemini 3 Pro High | Policy knowledge |
| Blog/content writing | Gemini 3 Pro High | Natural language |
| Web search / trends | Gemini 3 Pro High | Live web access |
| Bug fixing / debugging | Claude Opus 4.5 | Deep reasoning |
| Complex logic / algorithms | Claude Opus 4.5 | Analytical |
| Refactoring / architecture | Claude Opus 4.5 | Multi-file context |
| Tests / test fixing | Claude Opus 4.5 | Precise logic |
| Security / privacy | Claude Opus 4.5 | Detail-oriented |
| Database / queries | Claude Opus 4.5 | Structured thinking |

## Default
When unsure → **Claude Opus 4.5** (handles most coding tasks well)
