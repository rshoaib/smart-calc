# SEO Audit Workflow (Use Gemini 3.1 Pro)

## Step 1: Technical SEO
1. Check sitemap.xml exists and is submitted to GSC
2. Verify robots.txt allows crawling
3. Check all pages have unique title tags and meta descriptions
4. Verify proper heading hierarchy (single H1 per page)
5. Check for semantic HTML5 elements
6. Verify Canonical Tags (`<link rel="canonical" href="..." />`) exist and self-reference
7. Check for Orphan Pages (pages in sitemap without internal links)

## Step 2: Core Web Vitals
8. Run Lighthouse audit on key pages
9. Check CLS (Cumulative Layout Shift) < 0.1
10. Check LCP (Largest Contentful Paint) < 2.5s
11. Check FID (First Input Delay) < 100ms
12. Fix any flagged issues

## Step 3: Content SEO
13. Research target keywords with search volume data
14. Optimize existing content for target keywords
15. Add internal links between related pages
16. Add cross-site footer links (More Tools section)
17. Add structured data (JSON-LD) for rich snippets

## Step 4: Indexing
18. Check Google Search Console for indexing issues
19. Submit new/updated URLs for indexing
20. Monitor coverage report for errors
21. Check mobile usability report

## Step 5: AdSense Compliance (for ad-supported sites)
22. Verify ad placement follows AdSense policies
23. Check ad units are standard sizes (728x90, 300x250)
24. Ensure ads are not too close to interactive elements
25. Verify no hidden or obscured ads
