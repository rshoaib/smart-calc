# SEO Audit Workflow (Use Gemini 3.1 Pro)

## Step 1: Technical SEO
1. Check sitemap.xml exists and is submitted to GSC
2. Verify robots.txt allows crawling
3. Check all pages have unique title tags and meta descriptions
4. Verify proper heading hierarchy (single H1 per page)
5. Check for semantic HTML5 elements

## Step 2: Core Web Vitals
6. Run Lighthouse audit on key pages
7. Check CLS (Cumulative Layout Shift) < 0.1
8. Check LCP (Largest Contentful Paint) < 2.5s
9. Check FID (First Input Delay) < 100ms
10. Fix any flagged issues

## Step 3: Content SEO
11. Research target keywords with search volume data
12. Optimize existing content for target keywords
13. Add internal links between related pages
14. Add cross-site footer links (More Tools section)
15. Add structured data (JSON-LD) for rich snippets

## Step 4: Indexing
16. Check Google Search Console for indexing issues
17. Submit new/updated URLs for indexing
18. Monitor coverage report for errors
19. Check mobile usability report

## Step 5: AdSense Compliance (for ad-supported sites)
20. Verify ad placement follows AdSense policies
21. Check ad units are standard sizes (728x90, 300x250)
22. Ensure ads are not too close to interactive elements
23. Verify no hidden or obscured ads
